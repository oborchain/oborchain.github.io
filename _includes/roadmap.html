Roadmap 2018

Q1 Q2 Q3 Q4

Q1

The OBORCHAIN platform launches on the Ethereum Mainnet. Users can fund or crowdfund open source issues on GitHub using the OBOR token. Developers can claim the OBOR token after theyâ€™ve successfully resolved the GitHub issue.

Q2

The FundRequest platform allows any ERC 20 token to fund Open Source Issues on GitHub. Claiming funding is extended with a staking mechanism. Meanwhile the first Open Source Projects are chosen to receive support from the FundRequest Ecosystem Incentivisation pool.

Q3

A dispute flow is added to the FundRequest platform which enables community collaboration to settle raised disputes. OBORCHAIN enables the SkillToken to measure a developer's success rate and reputation.

Q4

Next to crypto assets the OBORCHAIN platform allows funding in fiat currencies (USD, EUR,..) and adds premium features to the FundRequest platform, such as invoicing and adding a Service Level Agreement (SLA) to a request.
